## Module to calculate CRCs in PowerShell

[Cyclic Redundancy Checks](https://en.wikipedia.org/wiki/Cyclic_redundancy_check) are a common way to verify file integrity.


## Installation

The module is available [on the PowerShell Gallery](https://www.powershellgallery.com/packages/CRC) and can be installed with

```
Install-Module CRC
```
